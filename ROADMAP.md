# Buildus Care 기능 로드맵

## Phase 1 — 완료 (2026-05-06 ~ 2026-05-11)

### 핵심 기능

- 전체 주문 흐름 (견적 → 예약 → 결제 → 시공 → 완료)
- AI 사진 판정 (Claude Vision)
- Toss 결제 연동
- 관리자 운영 UI (주문/슬롯/기사/분석)
- 기사 모바일 웹앱
- 고객 주문 현황 확인

### 운영 기능

- 예약 취소/환불 정책 자동화
- 슬롯 cap 활성 기사 수 연동
- 관리자 기사 배정/취소
- 퍼널 이벤트 트래킹
- PWA + Safe Area 대응

### 성능 최적화

- API 캐시 (슬롯 30초, 서비스 1시간)
- 페이지네이션 (주문/진단 20건)
- 인덱스 추가
- fetch 병렬화
- loading skeleton

## Phase 3 — 예약/알림 자동화 (다음 작업)

### 예약 변경 1클릭

- 고객이 주문 상태 페이지에서 직접 날짜/슬롯 변경
- 슬롯 재검증 + 기사 유지/재배정 자동 판단
- PATCH /api/orders/:id/reschedule

### 시공 전 알림 자동화

- D-1, 당일 아침 고객 알림
- Vercel cron 또는 외부 스케줄러
- notifications 큐 → SMS/카카오 발송 연동
